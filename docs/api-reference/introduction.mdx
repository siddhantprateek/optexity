## Quick Reference

### Core Types

| Type | Description |
|------|-------------|
| `Automation` | Top-level container with URL, parameters, and nodes |
| `ActionNode` | Single action (interaction, extraction, assertion, etc.) |
| `ForLoopNode` | Iterate over a variable's values |
| `IfElseNode` | Conditional execution |
| `Parameters` | Input and generated variables |

### Action Types

| Category | Actions |
|----------|---------|
| **Interaction** | click, input, select, check, scroll, navigate, upload, download, agentic |
| **Extraction** | LLM, network call, Python script, screenshot |
| **Assertion** | LLM, network call, Python script |
| **2FA** | Email, TOTP, API call |
| **Misc** | Python script execution |

### Key Concepts

- **Locators**: Find elements using Playwright commands, XPath, or AI with `prompt_instructions`
- **Variables**: Use `{variable_name[index]}` syntax for dynamic values
- **Timing**: Control with `before_sleep_time`, `end_sleep_time`, `max_tries`
- **Loops**: Iterate with `ForLoopNode` using `{variable_name[index]}`

## Import Patterns

Common imports for building automations:

```python
# Core automation
from optexity.schema.automation import (
    Automation,
    ActionNode,
    ForLoopNode,
    IfElseNode,
    Parameters,
)

# Interaction actions
from optexity.schema.actions.interaction_action import (
    InteractionAction,
    ClickElementAction,
    InputTextAction,
    SelectOptionAction,
    CheckAction,
    ScrollAction,
    GoToUrlAction,
    GoBackAction,
    SwitchTabAction,
    CloseCurrentTabAction,
    CloseAllButLastTabAction,
    UploadFileAction,
    DownloadUrlAsPdfAction,
    AgenticTask,
    CloseOverlayPopupAction,
)

# Extraction actions
from optexity.schema.actions.extraction_action import (
    ExtractionAction,
    LLMExtraction,
    NetworkCallExtraction,
    PythonScriptExtraction,
    ScreenshotExtraction,
)

# Assertion actions
from optexity.schema.actions.assertion_action import (
    AssertionAction,
    LLMAssertion,
    NetworkCallAssertion,
    PythonScriptAssertion,
)

# 2FA actions
from optexity.schema.actions.two_factor_auth_action import (
    Fetch2faAction,
    FetchEmail2FaAction,
    FetchTotp2FaAction,
    Fetch2faApiCallAction,
)

# Misc actions
from optexity.schema.actions.misc_action import PythonScriptAction
```
