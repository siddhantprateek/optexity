This page documents all interaction action types and their parameters.

## InteractionAction

Container for all interaction types. Exactly one action type must be provided.

```python
from optexity.schema.actions.interaction_action import InteractionAction

InteractionAction(
    start_2fa_timer=False,
    max_tries=10,
    max_timeout_seconds_per_try=1.0,
    click_element=...,  # Exactly one of these
)
```

### Properties

<ParamField path="start_2fa_timer" type="bool" default="False">
  Start the 2FA timer when this action executes. Only valid with `click_element`.
</ParamField>

<ParamField path="max_tries" type="int" default="10">
  Maximum attempts to find the element. Reduced to 5 if `skip_prompt=True`.
</ParamField>

<ParamField path="max_timeout_seconds_per_try" type="float" default="1.0">
  Seconds to wait for element on each attempt.
</ParamField>

### Action Types

Exactly one must be provided:

| Property | Type | Description |
|----------|------|-------------|
| `click_element` | `ClickElementAction` | Click on an element |
| `input_text` | `InputTextAction` | Type text into a field |
| `select_option` | `SelectOptionAction` | Select from dropdown |
| `check` | `CheckAction` | Check/uncheck checkbox |
| `download_url_as_pdf` | `DownloadUrlAsPdfAction` | Save page as PDF |
| `scroll` | `ScrollAction` | Scroll the page |
| `upload_file` | `UploadFileAction` | Upload a file |
| `go_to_url` | `GoToUrlAction` | Navigate to URL |
| `go_back` | `GoBackAction` | Go back in history |
| `switch_tab` | `SwitchTabAction` | Switch browser tab |
| `close_current_tab` | `CloseCurrentTabAction` | Close current tab |
| `close_all_but_last_tab` | `CloseAllButLastTabAction` | Close all except last tab |
| `agentic_task` | `AgenticTask` | AI-controlled task |
| `close_overlay_popup` | `CloseOverlayPopupAction` | Dismiss popups |

## BaseAction Properties

These properties are shared by `ClickElementAction`, `InputTextAction`, `SelectOptionAction`, `CheckAction`, and `UploadFileAction`:

<ParamField path="xpath" type="str | None" default="None">
  XPath selector for the element. Mutually exclusive with `command`.
</ParamField>

<ParamField path="command" type="str | None" default="None">
  Playwright locator command (e.g., `get_by_role("button", name="Submit")`). Mutually exclusive with `xpath`.
</ParamField>

<ParamField path="prompt_instructions" type="str" required>
  Natural language description for AI fallback when locators fail.
</ParamField>

<ParamField path="skip_prompt" type="bool" default="False">
  Skip AI prompt if locator fails. Reduces `max_tries` to 5.
</ParamField>

<ParamField path="assert_locator_presence" type="bool" default="False">
  Only execute if element exists. Requires `command` to be set.
</ParamField>

## ClickElementAction

Click on an element.

```python
from optexity.schema.actions.interaction_action import ClickElementAction

ClickElementAction(
    command="""get_by_role("button", name="Submit")""",
    prompt_instructions="Click the submit button",
    double_click=False,
    expect_download=False,
    download_filename=None,
)
```

### Properties

Inherits all [BaseAction properties](#baseaction-properties), plus:

<ParamField path="double_click" type="bool" default="False">
  Perform a double-click instead of single click.
</ParamField>

<ParamField path="expect_download" type="bool" default="False">
  Whether this click triggers a file download.
</ParamField>

<ParamField path="download_filename" type="str | None" default="Auto-generated UUID">
  Filename to save the download. Only used when `expect_download=True`.
</ParamField>

## InputTextAction

Enter text into an input field.

```python
from optexity.schema.actions.interaction_action import InputTextAction

InputTextAction(
    command="""get_by_label("Email")""",
    input_text="{email[0]}",
    prompt_instructions="Enter the email address",
    is_slider=False,
    fill_or_type="fill",
)
```

### Properties

Inherits all [BaseAction properties](#baseaction-properties), plus:

<ParamField path="input_text" type="str | None" default="None">
  Text to enter. Supports variable substitution (`{var[0]}`).
</ParamField>

<ParamField path="is_slider" type="bool" default="False">
  Whether the element is a slider input.
</ParamField>

<ParamField path="fill_or_type" type="'fill' | 'type'" default="'fill'">
  How to enter text:
  - `fill`: Set value instantly (default)
  - `type`: Type each character (for autocomplete fields)
</ParamField>

## SelectOptionAction

Select values from a dropdown.

```python
from optexity.schema.actions.interaction_action import SelectOptionAction

SelectOptionAction(
    command="""get_by_label("Country")""",
    select_values=["United States"],
    prompt_instructions="Select the country",
    expect_download=False,
    download_filename=None,
)
```

### Properties

Inherits all [BaseAction properties](#baseaction-properties), plus:

<ParamField path="select_values" type="list[str]" required>
  Values to select. Supports variable substitution.
</ParamField>

<ParamField path="expect_download" type="bool" default="False">
  Whether selecting triggers a file download.
</ParamField>

<ParamField path="download_filename" type="str | None" default="Auto-generated UUID">
  Filename for download. Only used when `expect_download=True`.
</ParamField>

## ScrollAction

Scroll the page.

```python

    down=True,  # True = scroll down, False = scroll up
)
```

### Properties

<ParamField path="down" type="bool" required>

## AgenticTask

AI-controlled autonomous task.

```python
from optexity.schema.actions.interaction_action import AgenticTask

AgenticTask(
    task="Navigate to settings and enable notifications",
    max_steps=10,
    backend="browser_use",
    use_vision=False,
    keep_alive=True,
)
```

### Properties

<ParamField path="task" type="str" required>
  Natural language description of the goal. Supports variable substitution.
</ParamField>

<ParamField path="max_steps" type="int" required>
  Maximum number of actions the agent can take.
</ParamField>

<ParamField path="backend" type="'browser_use' | 'browserbase'" required>
  Agent backend to use.
</ParamField>

<ParamField path="use_vision" type="bool" default="False">
  Whether agent receives screenshots.
</ParamField>

<ParamField path="keep_alive" type="bool" default="True">
  Keep browser session alive after task completes.
</ParamField>

## CloseOverlayPopupAction

Specialized agentic task for dismissing popups.

```python
from optexity.schema.actions.interaction_action import CloseOverlayPopupAction

CloseOverlayPopupAction(
    max_steps=5,
)
```

### Properties

<ParamField path="task" type="str" default="Built-in popup dismissal prompt">
  Task description. Uses comprehensive default prompt.
</ParamField>

<ParamField path="max_steps" type="int" default="5">
  Maximum actions for popup dismissal.
</ParamField>

<ParamField path="backend" type="'browser_use' | 'browserbase'" default="'browser_use'">
  Agent backend.
</ParamField>

<ParamField path="use_vision" type="bool" default="True">
  Uses vision for popup detection.
</ParamField>

<ParamField path="keep_alive" type="bool" default="True">
  Keep browser session alive.
</ParamField>
